# tarefa_MatrizLedGrupo
O seguinte programa controla um mapa de LEDS 5x5 na Placa de Desenvolvimento BitDogLab gerando animações de acordo a comandos dispostos pelo terminal.
Comandos de 0 a 9, além dos comandos por *, #, A, B, C, D.
